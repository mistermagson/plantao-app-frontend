
# Plantao Juizes App - Frontend

Frontend para o APP Plantao Juizes

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/mistermagson/plantao-app-frontend
```

Entre no diretório do projeto

```bash
  cd plantao-app-frontend
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
