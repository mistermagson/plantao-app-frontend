# Projeto Plantao APP

## Frontend
Arquivos do frontend do projeto

### Usage

``` git clone https://github.com/mistermagson/plantao-app-frontend.git```

``` cd plantao-app-frontend/frontend```

### Instalar Dependencias

``` yarn install```

### Rodar Localmente

``` yarn dev ```

### Deploy

``` pm2 start yarn --name "frontend-plantao" -- run start ```
